```uml
@startuml

[*] --> トップページ
トップページ -> 会員情報 : ログイン情報を入力し、ログインをクリック
トップページ -> 会員情報 : 会員情報をクリック
トップページ ->商品一覧 : 商品一覧をクリック
トップページ -> カート : カートをクリック
会員情報 -> ログアウト : ログアウトするをクリック
会員情報 --> 会員情報編集 : 編集をクリック
会員情報編集 -> カート : カートをクリック
商品一覧 -> 商品検索 : 商品検索する
商品一覧 -> 商品詳細 : 商品をクリック
商品一覧 --> トップページ : トップページをクリック
商品一覧 --> カート : カートをクリック
商品検索 -> 商品詳細 : 商品をクリック
商品詳細 --> カート : カートに入れるをクリック
商品詳細 --> 商品一覧 : 前の画面に戻るをクリック
商品詳細 -> トップページ : トップページをクリック
カート -> 購入確認画面 : 注文をクリック
カート -> トップページ : トップページをクリック
購入確認画面 -> 購入完了 : 注文を確定するをクリック
購入確認画面 --> 会員情報編集 : お届け先が違う場合をクリック
購入完了 -> トップページ : トップページをクリック

@enduml

```
