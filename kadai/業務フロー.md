```uml

@startuml
opt 検索
ユーザー -> webサーバー : 検索（商品名）
webサーバー -> DBサーバー : 検索処理（商品名）
DBサーバー -> DBサーバー : 検索処理
DBサーバー -> webサーバー : 検索結果
webサーバー -> ユーザー : 検索結果
 end


opt ショッピングカートに入れる
ユーザー -> webサーバー : 数量入力
webサーバー -> DBサーバー : 在庫確認
DBサーバー -> DBサーバー : 在庫確認
DBサーバー -> webサーバー : 在庫結果
webサーバー -> ユーザー : 在庫結果

alt 在庫あり
 webサーバー -> ユーザー : 購入ボタン表示
else 在庫なし
 webサーバー -> ユーザー : 入荷日を表示
 end
end
@enduml

```
